# Postgraduate Degree in Software Engineering - Exercises
This repository aims to showcase the exercises completed during the Postgraduate Degree in Software Engineering. The exercises cover various topics related to software development and engineering practices.

## Contents
In this repository, you will find examples and solutions to exercises related to:

* Programming and application development
* Integration with APIs and web services
* Design patterns and software architecture
* Testing and quality assurance

## Repository Structure
The repository is organized into folders, each corresponding to a specific module or topic covered in the course. Within each folder, you will find solved exercises and examples related to the topic.

## How to Use
To use this repository:

1. Clone the repository or download the ZIP files.
2. Navigate to the folder of the topic of interest.
3. Read the README.md file in the topic folder, if available, for specific instructions on how to run and use the examples and exercises.

## Contribution
Feel free to contribute to this repository by providing additional solutions, bug fixes, or improvements to the existing examples and exercises. To contribute, fork the repository and create a pull request with your changes.
